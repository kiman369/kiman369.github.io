김기만 Kim Ki Man

정보보안 전문가를 꿈꾸는 대학생

소개

안녕하세요. 저는 2024년도에 세일고등학교를 졸업하고, 2025년도에 동양미래대학교 자유전공학과에 입학하여 현재 학업 중인 김기만입니다. 정보보안 전문가를 목표로 열심히 공부하고 있으며, BoB(Best of the Best) 프로그램에 도전할 계획입니다.

기술 스택

  • Unity (C#)
  • HTML, CSS, JavaScript (기초)
  • 보안 관련 기초 지식 학습 중

프로젝트 경험

로그라이크 덱 빌딩 게임 개발

Unity를 활용하여 로그라이크 형식의 덱 빌딩 게임을 직접 기획하고 개발하였습니다. 게임의 카드 시스템, 전투 로직, UI 등을 구현하였고, 반복 가능한 플레이를 고려한 구조를 설계했습니다.

목표

정보보안 전문가로 성장하여 BoB 프로그램을 통해 실무 경험과 고급 기술을 습득하고, 사이버 보안 분야에서 사회에 기여하고 싶습니다.
